## Deploy options (in order of fastest to ship)
- **Cloudflare Pages** — drag-and-drop the folder; free; custom domain in minutes.
- **Netlify Drop** — same idea, free, instant URL.
- **Vercel** — same.
- **GitHub Pages** — push to a repo, point Pages at it.

All four host static HTML for free. None require a build.
